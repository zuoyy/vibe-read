# 欢迎来到雪球部落！一份帮你快速上岸的新手指南

你好，勇敢的新朋友！

我们深知，**“负债人最怕的不是‘没钱’，而是‘孤立无援’和‘深深的羞耻感’”**。这种感觉，我们懂。正因如此，雪球部落诞生了。我们的使命很简单：把原本痛苦、孤独的还债过程，变成一场你和队友们并肩作战的**“集体通关游戏”**。在这里，你不是一个人，我们陪你一起滚大希望的雪球，直到成功上岸！

## 1. 告别孤单：你的第一步，找到组织！

告别孤立无援的感觉，第一步就是找到你的战友。下面将引导你完成加入社区的关键几步。

### 1.1 领一只属于你的“小狗”，开始匿名之旅

为了彻底实现“去羞耻化”，当你进入部落时，系统会为你分配一个可爱的“小狗昵称”（比如：“乐观的拉布拉多”）。在这里，没有真实姓名，只有共同的奋斗目标。这为你创造了一个绝对安全的环境，让你能毫无负担地分享和求助。

### 1.2 如何选择你的第一个“部落”？

根据你的具体情况，选择一个最适合你的部落，是开启游戏的第一步。我们主要有三类部落：

- **“上岸预备营”**
    - 这是为刚刚踏上还债旅程的新手们准备的理想起点。在这里，你可以学习基础知识，找到节奏。
- **“网贷终结者”**
    - 这个部落专注于处理高频率、小金额的各类网贷。如果你正被这类债务困扰，这里的战友和策略会给你巨大帮助。
- **“房奴互助会”**
    - 为正在承受长期房贷压力的朋友们设立的专属支持小组。大家在这里分享经验，互相打气，共同度过漫长的还款岁月。

### 1.3 什么是“互助树洞”？

每个部落内部都有一个特别的、绝对安全的空间——“互助树洞”。这里是让你倾诉最私密、最紧急问题的地方。比如：

- “今天被催收了怎么办？”
- “如何跟家人开口说这件事？”

在树洞里，你能获得来自“过来人”最真诚和实用的建议。

找到你的部落后，真正的游戏就开始了！让我们看看如何与队友一起，把债务雪球滚起来。

## 2. 核心玩法：如何在这里“打怪升级”？

在雪球部落，还债不再是枯燥的数字，而是一系列有目标、有反馈、有队友的闯关活动。

### 2.1 参加“集体挑战赛”，见证滚雪球的力量

想象一下，当一个人独自还清 500 元时，他可能觉得微不足道；但当部落发起“百日消灭小额贷”挑战，成千上万的人一起打卡时，那种**“集体胜利的狂热”**会产生巨大的驱动力。

部落主页上有一个巨大的屏幕，实时显示着“此时此刻，部落成员已共同消灭了 XXX 万元债务”。这种强大的视觉反馈会让你真切地感受到：**我不是一个人在战斗！**

### 2.2 撰写“成功日记”，重建你的自信

你可以将每天的微小进步写成“成功日记”，并选择一键发布到“广场”。哪怕只是“今天坚持没点外卖，省下 20 元”这样的小事，都会收获来自社区的鼓励。

一个独特的设定是：在这里，点赞不叫“赞”，叫**“投喂骨头”**。这种低门槛、高频率的正向反馈，旨在帮助你快速重建在债务压力下被磨损的自尊心。

### 2.3 探索“情报交换站”，获取上岸干货

这是一个共享实战经验的宝库，为你提供上岸所需的“硬核装备”：

- **谈判模版库**
    - 在这里，你找到的不只是模版。你能获取到由社区成员验证过的“战报”（battle reports）——那些与特定平台成功协商还款的真实经历和完整话术。这不是理论，而是被验证过的实战策略。
- **副业避雷针**
    - 负债时最容易被“高收益副业”的骗局吸引。这里有社区共同建立的副业“红黑榜”，帮你有效识别和规避风险。
- **技能交换**
    - 你可以在这里发布自己的技能来换取帮助，比如用“我会写文案”换“你会做表格”，以极低的成本解决问题，甚至找到靠谱的合作机会。

通过参与这些核心活动，你不仅在还清债务，更在社区中扮演着越来越重要的角色。

## 3. 你在部落的角色：从新手到导师

在雪球部落，每个人都有机会从一个求助者成长为领导者。

|   |   |   |
|---|---|---|
|角色 (Role)|如何成为？ (How to Become?)|核心职责 (Core Responsibilities)|
|**成员 (Member)**|搜索并申请加入符合你债务类型的部落。|每日打卡还款进度、撰写成功日记、为队友“投喂骨头”（点赞）。|
|**部落长 (Tribe Chief)**|连续记录“成功日记”7天，或消耗“金骨头”。|制定部落宣言、审核成员、发起集体挑战、发布鼓励信息。|

成为“部落长”不仅仅是一种责任，更是你个人成长旅程中的一个关键里程碑——从寻求帮助到给予帮助。当你带领部落取得胜利时，你不仅能赢得荣誉勋章，更会成为一座希望的灯塔，用你的亲身经历，指引更多人走向属于他们的“上岸”时刻。

无论你是成员还是部落长，你和你的部落都将参与一场公平又激动人心的竞赛。

## 4. 公平的竞技场：排行榜系统详解

我们的排行榜系统经过精心设计，确保每一位成员，无论债务多少，都能在一个公平的环境中竞争和进步。

### 4.1 我们比什么？进度 vs. 意志力

我们不比拼谁还的钱多，而是比拼谁更努力、更坚持。排名主要基于两个核心维度：

- **还款进度比 (Repayment %)**
    - 计算公式为 `(已还金额 / 初始负债总额)`。这意味着，负债1万的人还了1千，和负债100万的人还了10万，在榜单上的进度是完全一样的。
- **意志力积分 (Willpower Score)**
    - 这个分数基于你的每日打卡、撰写成功日记的频率和社区互动情况。它奖励的是你的坚持和对他人的贡献。

### 4.2 不同阶段，不同赛道

为了避免新手被高级玩家“劝退”，我们将排行榜分成了不同的赛道。你只会和进度相近的人同场竞技：

1. **[初级] 碎冰榜 (0% - 10%)**：在这个阶段，核心是**迈出关键的第一步，并肩养成坚持的好习惯**。
2. **[中级] 奔跑榜 (10% - 50%)**：在这里，游戏的核心变为**保持节奏，并积极探索增加收入的途径**。你已找到感觉，现在是加速的时候了！
3. **[高级] 曙光榜 (50% - 90%)**：胜利的曙光就在眼前，这一赛道的目标是**全力冲刺，同时建立防止财务问题复发的防线**。
4. **[终极] 上岸墙 (100%)**：这是属于毕业生的荣誉殿堂，成功上岸的你将成为激励新人的荣誉导师。
